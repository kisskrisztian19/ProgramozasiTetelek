# ProgramozasiTetelek

## Elõkészítés
 - Tíz elemû, egész számot tartalmazó tömb létrehozása
 - A tömb elemei:
   - 12, 7, 2 ,4, 45, 88, 1, 8, 95, 50
  - A tömb az összes metódusban látható legyen
  - TombKiiras metódus létrehozása
    - Nem ad vissza értéket
    - Nincs paramétere
    - A tömb elemeit vesszõvel elválasztva a konzolra kiírja.
## Összegzés
  - Osszegzes metódus létrehozása
    - Összegzés tétele a tömbön
    - Eredmény kiírása a konzolra
## Megszámolás
  - Megszamolas metódus
    - Hány 3-al osztható elem van a tömbben
    - Eredmény kiírása a konzolra
## Eldöntés
  - Eldontes metódus
    - Van-e a tömbben 69
    - Eredmény kiírása a konzolra
